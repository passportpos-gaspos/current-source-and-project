Device Name

"Device Name" is the name which is listed under Bluetooth Devices in the Android Bluetooth setting screen, when pairing.

iOS Port Name

"iOS Port Name" is used in the iOS environment. When the Device Name is changed, the iOS Port Name is changed at the same time. However, the iOS Port Name can be changed independently.

Auto Connect

This function is for iOS device only. If you want to connect your Star POS device to a Android device, please do not turn ON the Auto Connect function.

Security

If the Auto Connect is ON, the security setting should be "SSP".

Complete

After you change settings using this utility, please turn the Star Bluetooth device OFF and ON to apply settings.

Notice

In case you cannot input device name or PIN code correctly, try to change Input Method setting of Android itself.
Recommended setting: Android keyboard
